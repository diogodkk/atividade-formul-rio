Questionário

1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?

O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido.

2. Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Esse é um recurso realmente útil de rótulos de formulários HTML que facilitam o clique na opção desejada.

3. Na tag input para que serve o atributo "value"?

Deve usar o atributo value para definir o valor a ser enviado por este item.

4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Para usar este tipo de campo de um formulário, temos que fornecer o valor "radio" ao atributo type da tag input. Também devemos fornecer o atributo nome, de tal modo que todas as opções, todo o grupo de botões rádio, devem ter o mesmo o nome.

5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?

Um elemento HTML label representa uma legenda para um item em uma interface de usuário. Ele pode estar associado com um elemento de controle, colocando este dentro do elemento label , ou usando o atributo for. Logo o for tem que apontar para o ID do input.

6. Na tag option para que serve o atributo "selected"?

O elemento HTML select select representa um controle que apresenta um menu de opções. As opções dentro do menu são representadas pelo elemento option.

7. Nas tags de formulário para que serve o atributo "required"?

Usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo. Ao adicionar esse atributo a um campo de formulário, o navegador obriga o usuário a inserir dados naquele campo antes de enviar o formulário.

8. Na tag form para que serve o atributo "action"?

O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

9. Na tag form para que serve o atributo "method"?

O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST").

10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

POST não inclui o corpo na URL. Portanto, os dados enviados não ficarão visíveis na URL.

11. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

O valor POST é um recurso que permite realizar uploads de arquivos de texto e binários.
